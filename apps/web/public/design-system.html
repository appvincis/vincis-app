08 — Tokens & Referência Rápida para IA

Guia rápido para IA gerar telas

Fontes

Títulos/Display → font-serif (Newsreader)

UI / Corpo → font-body / font-label (Manrope)

Labels caps → text-[10px] font-bold uppercase tracking-widest

Ativo em nav → font-serif italic

Superfícies (mais escuro = mais elevado)

Fundo global → bg-background (#fdf9f6)

Cards base → bg-surface-container-lowest

Sidebar → bg-surface-container-low

Hover/pressed → bg-surface-container-high

Bordas & Radius

Divisores sutis → border-outline-variant/10 ou /15

Cards → rounded-xl (0.75rem)

Seções → rounded-2xl (1rem)

Pills/tags → rounded-full

Timeline items → rounded-[10px]

Interação

Press → active:scale-95 transition-all

Focus ring → focus:ring-2 focus:ring-primary/50

Hover texto → hover:text-primary transition-colors

Hover bg → hover:bg-surface-bright transition-all

Icons → Material Symbols Outlined

Icon ativo → FILL=1
